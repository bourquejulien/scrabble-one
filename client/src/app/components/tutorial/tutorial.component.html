Tutoriel
  • Un clique gauche sur l'une des cases du tableau de jeu permet d'écrire votre mot horizontalement à partir de votre clavier. Un deuxième clique sur la case permet d'écrire le mot verticalement.
  • Un clique droit sur une ou plusieurs lettres du chevalet permet d'effectuer un échange avec la réserve lorsque c'est votre tour.
  • Un clique gauche permet de sélectionner la lettre à déplacer sur le chevalet en utilisant les flèches droite et gauche du clavier.
  • La case blanche vous permet d'écrire un mot contenant une lettre que vous ne possédez pas. Il vous suffit simplement de remplacer la lettre désirée par une majuscule.
  • Pour échanger une lettre blanche, vous pouvez écrire la commande !échanger * dans la boite de communication.
  • Le bouton skip_next permet de passer son tour.
  • Le bouton question_answer permet de basculer le clavardage.
  • Les boutons remove_circle_outline et add_circle_outline permettent de modifier la taille du texte sur le plateau de jeu.
